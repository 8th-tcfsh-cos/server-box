50X Server Error

伺服器爆炸了。請稍候重試，回到首頁或是聯絡管理員。

發生了什麼事？

如果不是因為我們的伺服器是馬鈴薯串成的，就是以下某一件事發生了：

  1. 500 Internal server error   我也不知道發生了什麼
  2. 501 Not implemented   你傳送了 server 看不懂的要求。
  3. 502 Bad gateway; 504 Gateway Timeout   你連線到了 gateway 但是他覺得伺服器爆了。
  4. 200 OK   你在網址列輸入了 https://cos.tcfsh.tk/50x-error.html 。
  5. 503 Service unavailable   字面上的意思
  6. 501 Insufficient storage   也是字面上的意思 (我絕對沒有懶得打)

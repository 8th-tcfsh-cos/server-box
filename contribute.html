幫助我們

你希望台中一中的第八屆科學班成發有自己的網站嗎？你想讓之後每屆科學班都能有自己的網站嗎？那就加入我們的合資，讓 TCFSH COS 有自己專屬的 web server (還有網域) 吧！

為什麼要這樣做？

我之前做的成發箱子是發佈在 Github Pages 上的靜態網頁，但是作為一個箱子沒有後端是很詭異的事 (?)，於是做了一個有 PHP 後端的版本。目前此版本的箱子改為架設於台灣的 Google Cloud VPS 上，但出現了各種更新不方便、效能不足等問題。我非常希望能購買自己的主機，這樣不但更新迅速 (我就睡在他旁邊)，而且效能比起各大 web hosting 服務也是...呃...屌虐 (?

And by 屌虐 I mean

箱子
感謝武峰貢獻了他的電腦當伺服器，我們現在有新的 Web server 了！(詳情請見714)

這裡是台中一中第八屆科學班的官方網站，簡稱箱子 (我也不知道為什麼叫箱子)。在架設網站的過程中，我們遇到各種風雨及困境 (?) 還有各種屁孩、僵屍網路與菜雞駭客的無效攻擊 (而且有點小多)。在少數人大家的支持下，我們終於完成了多種功能的班級網站。有任何 feature request 都可以直接告訴開發者喔~

還有各種無效的連線請求基本上都會被我用 404 敷衍。這裡是 Debian 裡的 nginx，不是什麼爛 Windows Server 或大陸製的路由器系統，也不會是你一個 XSS 或 XML entity 可以簡單打穿的東西。既然你有時間、能力寫 injection script 或做出僵屍網路自動攻擊，何不拿來做些有意義的事呢？ :)
